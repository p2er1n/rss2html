# rss2html

纯前端 RSS/Atom 订阅源转 HTML 页面工具。

## 功能

- 解析 RSS 2.0 和 Atom 订阅源
- 生成简洁的静态 HTML 页面
- 支持自定义标题
- **纯前端实现，无需后端**

## 在线使用

https://p2er1n.github.io/rss2html/

## 本地使用

直接在浏览器打开 `index.html` 即可。

## 注意事项

由于浏览器同源策略（CORS）限制，本工具使用公共 CORS 代理获取订阅源内容。如果某个订阅源无法获取，可能是：

1. 代理服务暂时不可用（已配置多个代理自动切换）
2. 订阅源 URL 不正确或已失效
3. 订阅源服务器拒绝了代理请求
